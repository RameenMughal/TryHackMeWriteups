# Operating Systems Basics

Module: [Operating Systems Basics](https://tryhackme.com/module/operating-systems-basics)

Learn the fundamentals of operating systems and how users interact with Windows and Linux through graphical and command line interfaces.

Every device you use runs an operating system, it's the layer between you and the hardware. In this module, you’ll explore what an operating system is, its core duties, and common use cases. You will then focus on practical interaction with Windows through its graphical user interface, before moving to Linux and Windows command line environments to navigate and read files, and collect basic system information.
