# Software Basics

Module: [Software Basics](https://tryhackme.com/module/software-basics)

This module introduces the core building blocks behind software: data representation and encoding. You’ll learn how computers store colours, numbers, and text, and how character encoding turns letters into numeric values that systems can understand.
