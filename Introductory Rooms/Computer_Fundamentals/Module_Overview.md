# Computer Fundamentals

Module: [Computer Fundamentals](https://tryhackme.com/module/computer-fundamentals)

This module covers the fundamentals of how computers work. You'll start by exploring what's inside a computer, the components that make it tick and how they work together. From there, you'll discover the different types of computers and their specialisations, learn how systems communicate with each other through client-server relationships, and explore how virtualisation lets us create multiple virtual machines from a single computer. The module wraps up with cloud computing, showing how virtualisation scales up to power the services we use every day.
